# GB/T 学位论文格式规范

中华人民共和国国家标准 GB/T 7713.1-2006《学位论文编写规则》规定的学位论文格式要求。

## 页面设置

| 参数 | 要求 |
|------|------|
| 纸张大小 | A4 (210mm × 297mm) |
| 上页边距 | 3.0 cm |
| 下页边距 | 3.0 cm |
| 左页边距 | 3.0 cm |
| 右页边距 | 2.5 cm |
| 装订线 | 0 cm（通常包含在左边距内） |
| 版式 | 纵向 |

> 注：不同高校可能有细微差异，以上为常见通用值。

## 字体与字号

| 元素 | 中文 | 英文 | 字号 | 样式 |
|------|------|------|------|------|
| 正文 | 宋体 | Times New Roman | 小四（12pt） | 常规 |
| 一级标题 | 黑体 | Arial / Times New Roman | 三号（16pt） | 加粗 |
| 二级标题 | 黑体 | Arial / Times New Roman | 四号（14pt） | 加粗 |
| 三级标题 | 黑体 | Arial / Times New Roman | 小四（12pt） | 加粗 |
| 页眉 | 宋体 | Times New Roman | 五号（10.5pt） | 常规 |
| 页码 | 宋体 | Times New Roman | 五号（10.5pt） | 常规 |
| 图题/表题 | 宋体 | Times New Roman | 五号（10.5pt） | 加粗 |
| 参考文献 | 宋体 | Times New Roman | 五号（10.5pt） | 常规 |
| 脚注 | 宋体 | Times New Roman | 小五（9pt） | 常规 |

## 段落格式

| 参数 | 要求 |
|------|------|
| 行间距 | 1.5 倍行距（固定值 22pt 也可） |
| 段前间距 | 0 行 |
| 段后间距 | 0 行 |
| 首行缩进 | 2 字符 |
| 对齐方式 | 两端对齐 |

### 特殊段落

| 段落类型 | 特殊要求 |
|----------|----------|
| 标题 | 段前 1 行，段后 0.5 行，居中或左对齐，无首行缩进 |
| 参考文献 | 悬挂缩进 2 字符 |
| 图题/表题 | 居中，无首行缩进 |
| 公式 | 居中 |

## 标题层级

1. **一级标题**：第1章 / Chapter 1（三号黑体加粗，居中）
2. **二级标题**：1.1（四号黑体加粗，左对齐）
3. **三级标题**：1.1.1（小四黑体加粗，左对齐）
4. **四级标题**：1.1.1.1（小四楷体/宋体，左对齐，通常不推荐再深）

## 页眉页脚

| 元素 | 要求 |
|------|------|
| 页眉文字 | 奇数页：学校名称+学位论文标题；偶数页：章标题 |
| 页眉字体 | 宋体五号，居中 |
| 页码位置 | 页脚居中 |
| 页码格式 | 阿拉伯数字（1, 2, 3...）；目录/摘要部分用罗马数字（i, ii, iii...） |

## 图和表

- 图序与图题居中置于图下方，五号宋体
- 表序与表题居中置于表上方，五号宋体加粗
- 图/表编号按章节（如图 1-1、表 2-3），或全文连续编号

## 参考文献（GB/T 7714-2015）

GB/T 7714-2015 "参考文献著录规则" 定义了中文论文参考文献格式。

### 常见文献类型标识

| 类型 | 代码 |
|------|------|
| 普通图书 | [M] |
| 期刊文章 | [J] |
| 学位论文 | [D] |
| 会议论文 | [C] |
| 专利 | [P] |
| 标准 | [S] |
| 报纸文章 | [N] |
| 电子资源 | [EB/OL] |
| 报告 | [R] |

### 著录格式示例

**期刊文章**：
> 作者. 题名[J]. 刊名, 年, 卷(期): 起止页码.
> 张三, 李四. 人工智能在医学中的应用[J]. 计算机学报, 2023, 46(3): 512-530.

**图书**：
> 作者. 书名[M]. 出版地: 出版社, 年.
> 王五. 机器学习导论[M]. 北京: 清华大学出版社, 2020.

**学位论文**：
> 作者. 题名[D]. 保存地点: 保存单位, 年.
> 赵六. 基于深度学习的图像分割研究[D]. 北京: 北京大学, 2022.

**会议论文**：
> 作者. 题名[C]//会议名称. 会议地点, 年: 起止页码.
> Smith J. Deep learning advances[C]//Proc of NeurIPS. Vancouver, 2023: 1234-1245.

### 文中引用格式

GB/T 7714 采用**顺序编码制**：
- 文中引用：上标角注，如 `[1]`、`[1,3]`、`[1-5]`
- 参考文献列表按引用顺序排列
